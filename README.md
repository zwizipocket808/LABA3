Задание 1
Условие: 
Реализовать функцию для поиска значения в сложном объекте (произвольно вложенные списки). Функция должна возвращать индекс первого вхождения искомого значения в уплощённом списке (обход в глубину). Если значение не найдено – возвращать `None`. Требуется написать две версии: с использованием рекурсии и без неё.

Описание проделанной работы: 
Для решения задачи были созданы две функции:  
- `find_recursive(obj, value)` – рекурсивный обход: функция `flatten` рекурсивно раскрывает все вложенные списки в плоский список, затем ищет индекс значения методом `index()`.  
- `find_iterative(obj, value)` – итеративный обход с использованием стека (список `stack`): элементы извлекаются из стека, если элемент является списком – его содержимое помещается обратно в стек в обратном порядке для сохранения порядка обхода; если элемент не список – добавляется в плоский список. После построения плоского списка выполняется поиск индекса.  
Обе функции возвращают индекс первого вхождения или `None`. Проверка выполнена на примерах из задания.

Задание 2
Условие:
Разработать функцию для вычисления последовательности  


Описание проделанной работы:  
Созданы две функции:  
- `recurrence_recursive(u, v, n)` – рекурсивная версия. Базовый случай: `n == 1` возвращает `(u, v)`. Рекурсивный случай: получает `(a_prev, b_prev)` для `n-1`, затем вычисляет `a = 2*b_prev + a_prev` и `b = 2*b_prev*b_prev + b_prev`.  
- `recurrence_iterative(u, v, n)` – итеративная версия. Переменные `a, b` инициализируются как `u, v`. В цикле от `2` до `n` выполняются те же формулы обновления.  
Обе функции корректно вычисляют значения для любых `n >= 1`. Для проверки использованы небольшие значения, например `u=1, v=2, n=3`.
